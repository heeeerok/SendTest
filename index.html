What we check

Automatic checks vs. human review

We automatically flag the embarrassing — and expensive — mistakes that slip through even experienced teams. Some errors require human judgment; we flag those too so nothing slips through before your subscribers see it.

✓ Caught automatically
✓
Duplicate URLs across different CTAs
Flags when two different buttons link to the same URL — the most common copy-paste mistake in multi-product campaigns
✓
Unresolved merge tags
Catches {{variable}}, *|MERGE_TAG|*, and ${var} left unreplaced in links or subject lines before your subscribers see them
✓
Staging / dev environment URLs
Detects staging., dev., qa., localhost and similar patterns that should never reach production sends
✓
Placeholder # links
Flags href="#" — these are not clickable in Outlook, Gmail, or Apple Mail and look broken to subscribers
✓
Missing unsubscribe link
Required by CAN-SPAM, GDPR, and most national anti-spam laws — sending without one is a legal and deliverability risk
✓
Subject line filename patterns
Catches internal filenames or date prefixes left as the subject — e.g. 20251225_CampaignName.html instead of your actual subject line
✓
Missing image alt text
Flags functional images without alt text — critical for accessibility and when images are blocked in email clients
→ Requires human review
→
Single wrong URL that is unique and live
If a CTA links to the wrong page but the page actually loads, the tool cannot detect it — always verify destinations manually before sending
→
URL leads to wrong destination page
A valid URL pointing to the wrong product or landing page requires live page title verification — coming in Pro
→
Broken countdown timers
Dynamic content rendered as images or JavaScript — the tool sees your HTML only, not the final rendered state
→
Wrong region or location labels
A venue listed under the wrong city is a content error invisible to any automated HTML checker
→
Authentication-walled destinations
Login-required pages cannot be verified even with live HTTP checks — manual review required

How it works

Clean sends in 60 seconds

No account. No install. Works with any ESP.

STEP 01
Export your email HTML
In your ESP, find the option to view or export the rendered HTML of your campaign.

In Klaviyo: Preview → </> HTML
In Mailchimp: Preview → Edit HTML
In Salesforce MC: View Source in preview mode
STEP 02
Paste and run
Paste the HTML into the box above. Optionally add your subject line and production domain for the most complete report.

Click Run QA Check. Results appear in under 1 second.
STEP 03
Fix and send with confidence
Review the report. Fix any critical issues flagged in red. Re-run to confirm everything is clean.

When the verdict shows Clear to Send — you're good to go.
